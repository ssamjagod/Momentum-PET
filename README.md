## :pencil2: 주요 기능
**# 반려견 홈페이지**
- 사용자는 반려견 홈페이지의 정보를 볼 수 있습니다.
- 사용자는 회사소개,팀원정보,회사위치를 볼 수 있습니다.
- 입양하기 화면에서 입양하고 싶은 반려견을 정하고 신청할 수 있습니다.
- 반려견 리스트에서 버튼을 클릭하여 찜한 반려견리스트에 담을 수 있습니다.
